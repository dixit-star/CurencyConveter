
# Currency Converter Web App

This is a **simple Currency Converter web application** that allows users to convert one currency into another.  
This project is built for **learning and practice purposes**, especially for React beginners.

**Live Demo:**  
https://curency-conveter.netlify.app

---

##  Features
- Enter amount to convert
- Select source currency
- Select target currency
- Instant conversion result
- Simple and clean user interface

---

##  Tech Stack
- HTML  
- CSS  
- JavaScript  
- React.js  
- Netlify (Deployment)
